# Fitness Tracker App (React Native)

This project contains the React Native implementation of the Fitness Tracker UI.

## Setup

1.  Make sure you have Node.js installed.
2.  Open this directory in your terminal:
    ```bash
    cd react_native_fitness
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Run the app:
    ```bash
    npx expo start
    ```

## Structure

- `App.js`: Main entry point.
- `src/components/`: UI components (Header, BodyOverview, MealPlan, etc.).
- `src/constants/`: Theme (Colors, Fonts).

## Notes

- Uses `expo-status-bar` and `@expo/vector-icons`.
- Standard `StyleSheet` is used for styling to match the provided Tailwind design.
